## dungeon-poc

Just PoC dungeon game used patterns of programming. For see patterns, type `pattern::::` in search input IDE

### local presentation:

* dungeon package
 - ```cd ./dungeon```
 - ```npm i```
 - ```npm run test```
 - ```npm run build```

* client
 - ```cd ./client```
 - ```npm i```
 - ```npm run dev```

### Then open http://localhost:${generated-port}
